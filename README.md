# stomp
## 0.概述
STOMP是一个基于帧的协议，它的帧是在HTTP上建模的。一帧由一个命令、一组可选的标题和一个可选的正文组成。STOMP是基于文本的，但也允许传输二进制消息。STOMP的默认编码是UTF-8，但是它支持消息正文的替代编码规范。
